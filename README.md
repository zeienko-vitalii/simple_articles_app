# Articles app

This project as a demonstration of the GraphQL use in the Flutter project and it comes with full description of step-by-step guide on **[Medium](www.google.com)**.

# Build and run 

```sh

flutter pub get
chmod +x update_schema.sh && ./update_schema.sh
flutter run

```
